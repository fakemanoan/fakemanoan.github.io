hKtweaks_7420

hKtweaks is an app originally forked from MTweaks. It is a somewhat advanced way to control various things on your device, like CPU governors, clock speeds, voltages and others. It requires root access to function (Magisk, KernelSU or otherwise).

hKtweaks_7420 is hKtweaks, but modified to support Exynos 7420 devices a bit better, as this was a program originally designed for S7, S8 and S9 devices.

Features:

  • CPU and GPU Voltage control
  • Battery controls
  • GPU adjustments
  • Int, Mif, Disp, Cam voltage adjustments
  • MoroSound v2.1.1 … among others
